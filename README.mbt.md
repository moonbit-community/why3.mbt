# why3.mbt

`why3.mbt` is a MoonBit implementation of a deliberately bounded WhyML-to-Z3
verification pipeline. The implementation target and compatibility gates are
specified in [`plan.md`](plan.md); the current parser baseline tracks Why3
1.7.2 at commit `1343338d3bb1941c0d4f134283bb0790816113c4`.
